[Vue Bits](https://vue-bits.dev/) 是一个Vue3的动画库，很多动画特效还是很好看的。

<!-- more -->

### 使用

#### 1. 全局安装 jsrepo

```bash
npm install jsrepo -g

# 或
pnpm add -g jsrepo
```

#### 2. 安装对应的组件

```bash
npx jsrepo add XXX

# 或
pnpm dlx jsrepo@latest add XXX

# 或
jsrepo add XXX
```

#### 3. 根据官方文档教程，使用组件即可
