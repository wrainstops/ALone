# ALone

Turborepo
A Turborepo, write my personal website and create some fun things.

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev

pnpm dev:3d
pnpm dev:blog
pnpm dev:frontplay
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```
